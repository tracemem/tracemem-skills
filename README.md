# TraceMem Skills

This repository contains skill definitions for using TraceMem correctly and safely with AI agents.

These skills are designed for use with agent frameworks that support skill files,
including https://skills.sh/.

## What These Skills Are
- Declarative instructions for agents
- Encode correct TraceMem usage patterns
- Emphasize safety, approvals, and governance

## What These Skills Are Not
- SDK documentation
- API reference
- Marketing material

## How to Use
Agents should load the skill files in `skills/` as part of their instruction set.

You may:
- Use all skills
- Select a subset
- Fork and customize

## Versioning
Skills evolve over time as TraceMem evolves.
See the VERSION file for the current skill set version.
